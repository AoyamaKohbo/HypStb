Popovの超安定論

Popovの超安定論は図に示す線形定常ブロックと非線形時変ブロックで構成されるフィードバックシステムが次の二つの条件を満足するとき、そのフィードバックシステムは安定(超安定)であることを示しています。

・線形ブロックの伝達関数が強正実条件を満足する。

・Popovの積分不等式が成立する。

強正実条件:

線形ブロックの伝達関数\(G(s)\)が次の条件を満足することです。(\(s\)はラプラス変数です。)

(1)\(G(s)\)が実のsに対して実である。

(2)\(G(s)\)が漸近的に安定、すなわちs面の右半平面\(Re[s]\ge0\)に極がない。

(3)\(-\infty <\omega< \infty\)のすべての\(\omega\)に対して\(Re[G(j\omega)]\ge0\)。 (\(j\)は虚数単位、\(\omega\)は角振動数です。)

Popovの積分不等式:

線形ブロックの出力を\(v\)、非線形ブロックの出力を\(w\)として、Popovの積分不等式は次式で与えられます。

\begin{align} \int_{0}^{t}vw dt \ge -\gamma_0^2 \end{align}

ここに、\(\gamma_0\)は定数です。

それでは、具体的なフィードバックシステムについて考察してみましょう。

入力\(u\)、出力\(x\)とする未知定数\(a(> 0)\)含む下記の実システムを想定します。

\begin{align} \dot x + a x = u \label{eq:RealSys} \end{align} ここでは、超安定論に従って未知定数\(a\)を同定する方法を見つけることを目的とします。 まず、このシステムに対して下記の規範モデルを与えます。 \begin{align} \dot x_M + \hat a x_M = u \label{eq:RefMdl} \end{align}

ここに、\(x_M\)は規範モデルの出力、\(\hat a (> 0)\)は定数です。規範モデルは計算機の中の数値計算モデルなので係数\(\hat a\)は調整可能です。 そこで、規範モデルの係数\(\hat a\)を調整して、実システムの\(a\)と一致させる方法を見つけましょう。 両者が一致すれば、\(a\)は\(\hat a\)として同定されます。

まず、式\eqref{eq:RefMdl}の両辺から式\eqref{eq:RealSys}の両辺をそれぞれ引きます。

\begin{align*} (\dot x_M - \dot x) + \hat a x_M - a x &= (\dot x_M - \dot x) + \hat a x_M - \hat a x + \hat a x - a x\\ &= (\dot x_M - \dot x) + \hat a (x_M - x) + (\hat a - a) x\\ &= 0 \end{align*}

次に\(v=\dot x_M - \dot x\)、\(\tilde a =\hat a - a\)とおくと、次式になります。

\begin{align} \dot v + \hat a v = -\tilde a x \label{eq:ErrSys} \end{align}

式\eqref{eq:ErrSys}の左辺を定常線形ブロック、右辺を非線形時変ブロックに分け、フィードバックシステムを構成します。

\(-w\)を入力、\(v\)を出力とする線形定常ブロック：

\begin{align} \dot v + \hat a v = -w \end{align}

\(x\)を入力、\(w\)を出力とする非線形時変：

\begin{align} w = \tilde a x \end{align}

これで、Popovの超安定論を適用するための線形定常ブロックと線形定常ブロックを構成し、互いの入出力関係を確認できました。 まず、Popovの積分不等式を満足させましょう。

Popov積分:

\begin{align} \int_{0}^{t}vw dt = \int_{0}^{t}\tilde a x v dt \end{align}

Popovの積分不等式を満足させるために次式を与えます。

\begin{align} \dot{\tilde a} = \alpha x v \label{eq:ModPara} \end{align}

すると、Popovの積分不等式が満足されることを次式で確認できます。

\begin{align*} \int_{0}^{t}vw dt = \frac{1}{\alpha}\int_{0}^{t}\dot{\tilde a} \tilde a dt = \frac{1}{2\alpha}[\tilde a^2]_0^t \ge -\frac{1}{2\alpha}\tilde a(0)^2 \end{align*}

次に強正実条件を確認します。 \(v\)、\(w\)のラプラス変換をそれぞれ\(V(s)\)、\(W(s)\)とすると、 \(v\)から\(w\)への伝達関数は次式となります。

\begin{align} G(s) = -\frac{V(s)}{W(s)} = \frac{1}{s+\hat a} \label{eq:TrnsFnc} \end{align}

実数のsに対してG(s)は実数となり、極は(-\(\hat a,~0)\)なのでs面の右半平面\(Re[s]\ge0\)に極はありません。 したがって、強正実条件(1)、(2)を満足します。 式\eqref{eq:TrnsFnc}の\(s\)に\(j\omega \)を代入します。

\begin{align*} Re[G(j\omega)] = Re[\frac{1}{\hat a + j\omega}] = Re[\frac{\hat a - j\omega}{\hat a + j\omega}] = \frac{\hat a}{\omega^2+\hat a^2} > 0 \end{align*}

強正実条件(3)を満足することが確認できました。

\(a\)が定数であることに注意すれば、式\eqref{eq:ModPara}の\(\dot{\tilde a}=\dot{\hat a}\)です。 そこで、次式が成立するように規範モデルの\(\hat a\)を調整します。

\begin{align} \dot{\hat a} = \alpha x v \end{align}

ここに、\(\alpha\)は定数です。すると、Popovの超安定条件が満足されるので次式が成立します。

\begin{align*} \lim_{t \rightarrow \infty} v = 0,~~~\lim_{t \rightarrow \infty} \hat a = a. \end{align*}

こうして漸近的に規範モデルは実システムと一致し、実システムの中の未知定数\(a\)は規範モデルの\(\hat a\)として同定されます。
